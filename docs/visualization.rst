Visualization
=============
Plots the attributes of the animals, the number of
herbivores and carnivores on the island, as well as
a heat map which shows the density of the herbivores
and carnivores in each cell.

This module was heavily inspired by:
https://github.com/heplesser/nmbu_inf200_june2020/
blob/master/examples/randvis_project/randvis/
simulation.py
